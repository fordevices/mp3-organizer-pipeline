WinAMP Speakers v2.0 beta 1 for WoodAmp
=======================================

This is a customization of the WinAmp speakers, to make
them blend in with my WoodAmp Skin. The only thing I did
was to change the bitmaps a little (well...). "Ray" is
the one to credit for the bulk of the work, and I've
included his text below.

Please note that for WinAmp 1.8+ the vis_wosp.dll should
be copied to the \Plugins directory.

For you all "design flaw seekers"; the bass speaker cone
in the normal sized speaker is reused as the treble speaker
cone in the double sized speaker. This made it a little
more difficult to blend in this particular speaker cone.

WoodAmp and the WoodAmp speakers is designed for 16 bit
color depth or more.

Please email me if you like or maybe dislike the WoodAmp
Skin: fambsson@hem.passagen.se

March 21, 1998,
Magnus Bengtsson.

And now for "Ray"s text:


This is a visualization plugin for WinAMP 1.6+ which adds speakers next to the
main program window.  Let me know what you think!

What's new in beta 1
--------------------
  - Better response
  - Speakers go into system tray for Winshade Mode

Installation
------------
  - You need WinAMP 1.6 or higher
  - Copy the enclosed vis_spkr.dll to your WinAMP directory
  - Go to the WinAMP Preferences dialog
  - Select the Plug-ins Tab
  - Select "WinAMP Speakers v2.0 beta 1" for the Visualization Library
  - Turn on "Auto execute on play"  
  - Play some music and you should see speakers to the right and left of WinAMP

Have fun!
